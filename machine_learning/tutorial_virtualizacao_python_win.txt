Para criar no windows um ambiente virtual Python:

1. python -m venv env_win

2. Verificar se há permissões de execução do script do ambiente virtual. Para isso abra, como administrador, o PowerShell e navegue até o diretório do
   ambiente virtual criado através do comando:
    
    2.1 Get-ExecutionPolicy

    Se a política de execução estiver definida como "Restricted" ou algo diferente de "RemoteSigned" ou "Unrestricted" usar os comandos:
   
    2.1 Set-ExecutionPolicy RemoteSigned (Uso não permanente)
    2.2 Set-ExecutionPolicy RemoteSigned -Scope CurrentUser (Uso permanente)

3. Para ativar e desativar o ambiente virtual:

   3.1. Entrar diretório do ambiente virtual, que geralmente é ~/<nome_do_ambiente_criado>/Script
   3.2 Usar o comando de ativação .\activate
   3.3 E usar o comando de desativação: .\deactivate
